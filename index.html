Welcome to Node of Life.

This demo is using the 'Game of Life' algorithm to play with a set of rules affecting each pixel, resulting in this 'organic like' grid. There is a nodejs server that sends the grid using a socket.io connection

To play just select your shape and click on the canvas to place that symbol and observe it's reactions
